Download Page

The latest QuantLib release is the 0.2.0 version. It includes the following files (Debian packages will be available shortly):

  1. QuantLib
    • Win32
      • QuantLib-0.2.0-full-inst.exe (7.87 MB): this is the main Win32 distribution file. It includes binaries, source code, documentation, and examples.
      • QuantLib-0.2.0-src.zip (815 KB): zipped source code.
      • QuantLib-0.2.0-light-inst.exe (1.50 MB): small installer with binaries only.
      • QuantLib-0.2.0-debug.zip (3.84 MB): debug libraries.
    • Unix
      • QuantLib-0.2.0-src.tar.gz (536 KB): gzipped source code.
      • QuantLib-0.2.0-src.tar.bz2 (416 KB): source code zipped with bzip2.
      • http://ftp.us.debian.org/debian/pool/main/q/quantlib: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)
  2. QuantLib Documentation
    • Win32
      • QuantLib-0.2.0-docs-html.zip (4.39 MB): zipped html documentation.
      • QuantLib-0.2.0-docs-refman.pdf (3.11 MB): pdf documentation.
      • QuantLib-0.2.0-docs-refman.pdf.zip (1.41 MB): zipped pdf documentation.
      • QuantLib-0.2.0-docs-refman.ps (5 MB): ps documentation.
      • QuantLib-0.2.0-docs-refman.ps.zip (668 KB): zipped ps documentation.
    • Unix
      • QuantLib-0.2.0-docs-html.tar.gz (2.67 MB): gzipped html documentation.
      • QuantLib-0.2.0-docs-html.tar.bz2 (1.72 MB): html documentation zipped with bzip2.
      • QuantLib-0.2.0-docs-refman.pdf (3.11 MB): pdf documentation.
      • QuantLib-0.2.0-docs-refman.pdf.gz (1.36 MB): gzipped pdf documentation.
      • QuantLib-0.2.0-docs-refman.pdf.bz2 (1.25 MB): pdf documentation zipped with bzip2.
      • QuantLib-0.2.0-docs-refman.ps (5 MB): ps documentation.
      • QuantLib-0.2.0-docs-refman.ps.gz (665 KB): gzipped ps documentation.
      • QuantLib-0.2.0-docs-refman.ps.bz2 (457 KB): ps documentation zipped with bzip2.
      • QuantLib-0.2.0-docs-man.tar.gz (81 KB): gzipped man documentation.
      • QuantLib-0.2.0-docs-man.tar.bz2 (51 KB): man documentation zipped with bzip2.
      • http://ftp.us.debian.org/debian/pool/main/q/quantlib-refman: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)
  3. QuantLib-Python
    The QuantLib-Python module misses an extended documentation. Anyway it includes a PyUnit test suite that shows the module usage.
    • Win32
      • QuantLib-Python-0.2.0.zip (186 KB): zipped source code.
      • QuantLib-Python-0.2.0.win32-py2.1.exe (518 KB): binaries distribution.
    • Unix
      • QuantLib-Python-0.2.0.tar.gz (114 KB): gzipped source code.
      • QuantLib-Python-0.2.0.tar.bz2 (80 KB): source code zipped with bzip2.
      • QuantLib-Python-0.2.0.linux-i686.tar.gz (1.54 MB): gzipped binaries.
      • QuantLib-Python-0.2.0.linux-i686.tar.bz2 (1.44 MB): binaries zipped with bzip2.
      • http://ftp.us.debian.org/debian/pool/main/q/quantlib-python: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)
