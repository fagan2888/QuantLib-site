Download Page

The latest QuantLib release is the 0.1.9 version. It includes the following files:

  1. QuantLib
    • Win32
      • QuantLib-0.1.9-src.zip: zipped source code.
      • QuantLib-0.1.9-inst.exe: executable installer.
      • QuantLib-0.1.9-debug.zip: debug libraries.
    • Unix
      • QuantLib-0.1.9-src.tar.gz: gzipped source code.
  2. QuantLib Documentation
    • Win32
      • QuantLib-0.1.9-docs-html.zip: zipped html documentation.
      • QuantLib-0.1.9-docs-refman.pdf: pdf documentation.
      • QuantLib-0.1.9-docs-refman.pdf.zip: zipped pdf documentation.
      • QuantLib-0.1.9-docs-refman.ps: ps documentation.
      • QuantLib-0.1.9-docs-refman.ps.zip: zipped ps documentation.
    • Unix
      • QuantLib-0.1.9-docs-html.tar.gz: gzipped html documentation.
      • QuantLib-0.1.9-docs-refman.pdf: pdf documentation.
      • QuantLib-0.1.9-docs-refman.pdf.gz: gzipped pdf documentation.
      • QuantLib-0.1.9-docs-refman.ps: ps documentation.
      • QuantLib-0.1.9-docs-refman.ps.gz: gzipped ps documentation.
      • QuantLib-0.1.9-docs-man.tar.gz: gzipped man documentation.
  3. QuantLib-Python
    The QuantLib-Python module misses an extended documentation. Anyway it includes a PyUnit test suite that shows the module usage.
    • Win32
      • QuantLib-Python-0.1.9.win32-py2.1.exe: binaries distribution.
      • QuantLib-Python-0.1.9.zip: zipped source code.
    • Unix
      • QuantLib-Python-0.1.9.tar.gz: gzipped source code.
      • QuantLib-Python-0.1.9.linux-i686.tar.gz: gzipped binaries.
