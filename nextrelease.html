Download Page

The latest QuantLib release is the 0.3.5 version (March 2004, CVS tag R000305f0-branch). It includes the following packages:

  1. QuantLib
  2. QuantLib documentation
  3. QuantLib Guile
  4. QuantLib MzScheme
  5. QuantLib Python
  6. QuantLib Ruby
  7. QuantLibXL (Excel add-in)

Previous QuantLib releases are available here.

  1. QuantLib
    • Win32
      • QuantLib-0.3.5-installer.exe (3636 kb): this is the main Win32 distribution file. It includes source code, tests, examples, and WinHelp documentation.
      • QuantLib-0.3.5.zip (1900 kb): zipped source code.
    • Unix
      • QuantLib-0.3.5.tar.gz (1273 kb): gzipped source code.
      • QuantLib-0.3.5-0.src.rpm (1268 kb): source RPM.
      • http://ftp.us.debian.org/debian/pool/main/q/quantlib: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)
  2. QuantLib Documentation
    • Win32
      • QuantLib-docs-0.3.5.chm (3184 kb): WinHelp documentation.
      • QuantLib-docs-0.3.5.pdf.zip (3430 kb): zipped pdf documentation.
      • QuantLib-docs-0.3.5.ps.zip (1133 kb): zipped ps documentation.
    • Unix
      • QuantLib-docs-0.3.5.html.tar.gz (2866 kb): gzipped html documentation.
      • QuantLib-docs-0.3.5.pdf.gz (3391 kb): gzipped pdf documentation.
      • QuantLib-docs-0.3.5.ps.gz (1090 kb): gzipped ps documentation.
      • QuantLib-docs-0.3.5.man.tar.gz (172 kb): gzipped man pages documentation.
      • QuantLib-docs-0.3.5-0.i386.rpm (7886 kb): RPM documentation.
      • http://ftp.us.debian.org/debian/pool/main/q/quantlib-refman: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)

    • Win32
