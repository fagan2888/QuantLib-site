Download Page

The latest QuantLib release is the 0.3.1 version (CVS tag R000300f1-branch). It includes the following packages:

  1. QuantLib
  2. QuantLib documentation
  3. QuantLib.NET
  4. QuantLib Guile
  5. QuantLib MzScheme
  6. QuantLib Python
  7. QuantLib Ruby

The Debian packages listed below will be available in a few days.

RQuantLib (which connects GNU R with QuantLib) is provided by Dirk Eddelbuettel at http://dirk.eddelbuettel.com/code/rquantlib.html

An initial Excel add-in is available as QuantLibXL in the QuantLib CVS and downloadable from http://quantlib.org/AutomatedBuilds/QuantLibXL/.

Previous QuantLib releases are available here while the last build performed on the current CVS snapshot can be downloaded here (only for leading edge QuantLib user and developers).

  1. QuantLib
    • Win32
      • QuantLib-0.3.1-full-inst.exe (12.6 MB): this is the main Win32 distribution file. It includes binaries, source code, documentation, and examples.
      • QuantLib-0.3.1-src.zip (930 KB): zipped source code.
      • QuantLib-0.3.1-light-inst.exe (1.7 MB): small installer with binaries only.
      • QuantLib-0.3.1-debug.zip (10 MB): debug libraries.
    • Unix
      • QuantLib-0.3.1-src.tar.gz (810 KB): gzipped source code.
      • http://ftp.us.debian.org/debian/pool/main/q/quantlib: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)
