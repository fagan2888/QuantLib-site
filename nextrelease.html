The latest QuantLib release is the 0.2.1 version. It includes the following files:

  1. QuantLib
    • Win32
      • QuantLib-0.2.1-full-inst.exe (11.6 MB): this is the main Win32 distribution file. It includes binaries, source code, documentation, and examples.
      • QuantLib-0.2.1-src.zip (890 KB): zipped source code.
      • QuantLib-0.2.1-light-inst.exe (1.6 MB): small installer with binaries only.
      • QuantLib-0.2.1-debug.zip (4.5 MB): debug libraries.
    • Unix
      • QuantLib-0.2.1-src.tar.gz (597 KB): gzipped source code.
      • QuantLib-0.2.1-src.tar.bz2 (457 KB): source code zipped with bzip2.
