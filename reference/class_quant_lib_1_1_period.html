Detailed Description

This class provides a Period (length + TimeUnit) class and implements a limited algebra.

Tests:
self-consistency of algebra is checked.
Examples:
BasketLosses.cpp, BermudanSwaption.cpp, Bonds.cpp, CallableBonds.cpp, CDS.cpp, ConvertibleBonds.cpp, CVAIRS.cpp, FittedBondCurve.cpp, LatentModel.cpp, Repo.cpp, and swapvaluation.cpp.
