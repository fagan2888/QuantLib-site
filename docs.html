Documentation

The QuantLib user documentation is not available, since QuantLib is not yet ready for the end user.

The documentation for the classes is available and is automatically generated from the source code using Doxygen. The documentation comes in different formats:

  • HTML: on-line browsing, zip, tar.gz
  • PDF: uncompressed, zip, gz
  • Postscript: uncompressed, zip, gz

Source code and Python module can be downloaded from the download page.

Feedback and questions concerning this site and project can be directed to
the QuantLib-users mailing list
or
the QuantLib administrator Ferdinando Ametrano.
