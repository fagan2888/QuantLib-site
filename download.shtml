Download Page

The latest QuantLib release is the 0.3.6 version (April 2004.) It includes the following packages:

  1. QuantLib
  2. QuantLib documentation
  3. QuantLib-Guile (bindings for Guile)
  4. QuantLib-MzScheme (bindings for PLT Scheme)
  5. QuantLib-Python (bindings for Python)
  6. QuantLib-Ruby (bindings for Ruby)
  7. QuantLibXL (Excel add-in)

Debian users can find version 0.3.6 of the different QuantLib packages in the "unstable" (a.k.a. "Sid") distribution; see the Debian site for a list of available packages. The previous release, 0.3.5, is still in the "testing" (a.k.a. "Sarge") distribution, but should be replaced by 0.3.6 in a few days.

For Mac OS X users, Fink packages are available in the "10.3/unstable" tree; for a list of available QuantLib-related files see the Package Database. To enable the "unstable" tree, please read the Fink faq. Be sure to run "fink selfupdate" to obtain the latest package definitions, and then run "fink install quantlib" to install the main library.

Packages for other platforms are available from the SourceForge download page in a number of different formats.
