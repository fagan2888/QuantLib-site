QuantLib-based Extension

QuantLib is already available as a Guile, MzScheme, Python, and Ruby modules by means of SWIG.
SWIG also supports Java, Tcl/Tk, Perl, and PHP, so this approach may easily help porting QuantLib to these languages. SWIG modules are available in the QuantLib CVS under QuantLib-SWIG. We welcome any contribution aimed at exporting the library towards other programming languages; however, the use of SWIG is to be preferred to hand-made approaches since it leads to more maintainable code.

QuantLib has been ported to the .NET framework in C#. See http://www.quantlib.net for more details.

There is a strong interest in having QuantLib available as extension for different platforms. We're looking for volunteers to help with:

  • COM for Microsoft world (VB and Excel);
  • Gnumeric add-in;
  • Matlab/Octave extension.
  • Mathematica extension.
  • S-Plus/R extension.
  • CORBA
  • Simple Object Access Protocol (SOAP)
  • XML financial instrument definition (FpML?)

Initial GNU R support is provided with RQuantLib by Dirk Eddelbuettel.

QuantLibXL is an initial Excel add-in built using xlw available in the QuantLib CVS.

Niels Elken Sønderby has a page for the tentative Mathematica project QuantLibMma.

In general the library should be exported to the maximum possible degree, allowing for object manipulation when possible, providing a function interface as bare minimum.

Feedback and questions concerning this site and project can be directed to the QuantLib-users mailing list or the QuantLib administrator Ferdinando Ametrano.
